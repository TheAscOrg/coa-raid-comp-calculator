# CoA Raid Comp Calculator

Hosted-ready CoA raid composition calculator built with Vite, React, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

The app is available from both `/` and `/coa_raid_comp.html`.

## Build

```bash
npm run build
```

Static icons and class art live in `public/assets`, so the production build copies them into `dist/assets`.

## Package Contents

- Packaged class icons: 21
- Packaged spec icons: 135
