# CoA Raid Comp Calculator

Standalone player export for the CoA raid composition calculator.

## Use

Open `coa_raid_comp.html` in a browser.

Keep the `assets` folder next to the HTML file. The page uses only local packaged icons and does not require the talent calculator repo at runtime.

## Package Contents

- Packaged class icons: 21
- Packaged spec icons: 50
- Missing spec icons: 5
